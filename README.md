# Comandos Linux para iniciantes

## Navegação

```bash
pwd - listar diretorio
ls - listar oq tem no diretorio
cd nome_da_pasta - ir para o diretorio desejado
cd .. - voltar 1 diretorio
```

## Criar arquivos e pastas

mkdir projetos
touch arquivo.txt

## Permissões
chmod 755 script.sh
chmod +x script.sh


## Atualização de pacotes
sudo apt update
sudo apt upgrade
